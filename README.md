# EventEmitter.js

My own interpretation of event management.

## Usage

List of methods:

 - `on`
 - `once`
 - `off`
 - `emit`

## Dependencies

- [underscore.js](http://underscorejs.org) 1.x+
- [underscore.assert.js](https://github.com/piecioshka/underscore.assert.js) 1.x+
